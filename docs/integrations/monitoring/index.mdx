# Monitoring & Alerting

Connect Koreshield to your monitoring stack to track security events, performance, and reliability.

## Available Integrations

- [Prometheus](prometheus.mdx)
- [Grafana](grafana.mdx)
- [Datadog](datadog.mdx)
- [Sentry](sentry.mdx)

## Guidance

- Use structured logs for reliable parsing and alerting.
- Track attack rates, provider errors, and latency.
- Create alerts for spikes in blocked requests and provider failures.
